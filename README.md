# RunInLife

App gamificado de atividades, conquistas e metas.

## Tecnologias
- Backend: Laravel (PHP)
- Frontend: Blade + JavaScript
- Banco de Dados: MySQL/MariaDB
- Estilo: CSS global (preto, vermelho, branco)

## Estrutura de Pastas
- `public/assets/css/global.css` — Estilo global
- `public/assets/js/app.js` — Scripts globais
- `resources/views/layouts/app.blade.php` — Layout base
- `resources/views/` — Views principais

## Funcionalidades
- Página inicial com progresso
- Gestão de hábitos, tarefas, conquistas, metas
- Ranking, loja, histórico
- Totalmente responsivo (desktop/mobile)

## Deploy
- Versionamento via Git
- Deploy na Hostinger
